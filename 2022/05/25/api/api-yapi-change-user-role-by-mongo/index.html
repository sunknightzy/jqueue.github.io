修改 YApi 的用户角色(添加管理员)

YApi 后端使用的是 MongoDB，先进入命令行管理工具：

mongo

然后进入 yapi 数据库：

use yapi

随后查询下指定用户的用户 id：

db.getCollection(“user”).find({“username”:”glmapper”})

也可以用邮箱来查：

db.getCollection(“user”).find({“email”:”glmapper_2018@163.com“})

查看返回数据中的“_id”字段的值，就是用户 id 了。

最后使用以下的语句更新指定用户的 role 为 admin，指定用户即可变成管理员用户：

db.getCollection(“user”).update({“username”:”glmapper”}, {$set: {“role”:”admin”}})

如果需要取消管理员权限，把 role 设置成 member 即可。
