ARK 插件基本规则及注意事项

SOFAARK 是一个轻量级的类隔离框架，其有两个基本的能力：解决依赖包冲突和多应用(模块)合并部署。本篇将从解决依赖角度来说明下 SOFARK 插件的基本使用规则。

下图是官方文档中提供的用于描述依赖包冲突的一个场景：

这里通过一个工程来模拟这种场景，然后通过将其中一个打包成插件的方式来解决。
